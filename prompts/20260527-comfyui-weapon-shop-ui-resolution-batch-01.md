# 20260527 ComfyUI Weapon and Shop UI Resolution Batch 01

## Goal

Generate original public-safe pixel art assets through local ComfyUI and publish exact `64x64` and `128x128` variants to the project home feed.

## Scope

- Workflow: `pokemon` / Z-Image local ComfyUI workflow
- Source generation size: `512x512`
- Final public PNG resolutions: `64x64`, `128x128`
- Categories: weapons, game shop UI
- Target pack size: 9 images per resolution/category pack
- Total public PNGs: 36
- Total feed WebP previews: 36
- Feed posts: 4 gallery posts

## Prompt Direction

Use original, non-branded fantasy RPG pixel art resource prompts. Keep silhouettes large and readable after downscaling, avoid readable text, avoid logos and watermarks, and keep all UI mockups icon-only.

## Categories

- Weapons: single centered equipment icons with strong silhouettes suitable for inventory use.
- Game Shop UI: square game store panel mockups with item grids, coin placeholders, buy/sell panel shapes, and no readable text.
