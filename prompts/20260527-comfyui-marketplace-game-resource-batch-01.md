# 20260527 ComfyUI Marketplace Game Resource Batch 01

## Goal

Generate marketplace-style public-safe pixel art game resource sets through local ComfyUI workflows.

## Scope

- Workflows: `qwen_image`, `pokemon`, `hidream_o1`
- Categories: characters, monsters, backgrounds, items, inventory UI
- Target pack size: 9 images per workflow/category pack
- Public curation target: `public/assets/<category>/20260527-comfyui-marketplace-game-resource-batch-01/<workflow>/<category>/`
- Feed preview target: `public/assets/feed-optimized/20260527-comfyui-marketplace-game-resource-batch-01/<workflow>/<category>/`

## Prompt Direction

Use original, non-branded pixel art resource prompts suitable for game ideation and marketplace-style asset previews. Keep visual designs public-safe, avoid readable text, avoid logos and watermarks, and prefer crisp silhouettes that can be evaluated quickly in a feed gallery.

## Categories

- Characters: playable hero concepts with full-body idle-ready poses.
- Monsters: enemy creature concepts with readable combat silhouettes.
- Backgrounds: wide side-view parallax-ready stage scenes.
- Items: single inventory object icons.
- Inventory UI: complete inventory panel mockups without readable text.
