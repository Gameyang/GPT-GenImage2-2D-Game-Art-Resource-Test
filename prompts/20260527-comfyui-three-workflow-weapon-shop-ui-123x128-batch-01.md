# 20260527 ComfyUI Three Workflow Weapon and Shop UI 123x128 Batch 01

## Goal

Generate one `123x128` weapon and game shop UI set from each local ComfyUI workflow.

## Scope

- Workflows: `qwen_image`, `pokemon`, `hidream_o1`
- Source generation sizes: workflow defaults (`1024x1024`, `512x512`, `768x768`)
- Final public PNG resolution: `123x128`
- Categories: weapons, game shop UI
- Target pack size: 9 images per workflow/category pack
- Total public PNGs: 54
- Total feed WebP previews: 54
- Feed posts: 6 gallery posts

## Prompt Direction

Use original, non-branded fantasy RPG pixel art resource prompts. Keep silhouettes large and readable after `123x128` export, avoid readable text, avoid logos and watermarks, and keep all UI mockups icon-only.

## Categories

- Weapons: single centered equipment icons with strong silhouettes suitable for inventory use.
- Game Shop UI: square source game store panel mockups exported to `123x128`, with item grids, coin placeholders, buy/sell panel shapes, and no readable text.
