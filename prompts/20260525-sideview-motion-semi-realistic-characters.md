## Goal

Regenerate four semi-realistic 2D game characters specifically for side-view gameplay and future motion production.

## Shared Prompt Constraints

- Use case: stylized-concept
- Asset type: side-view 2D game character base concept for animation planning
- Style: non-pixel semi-realistic 2D digital painting, readable side-scroller game art
- Camera: strict orthographic side profile, facing right, no perspective distortion
- Pose: neutral idle-ready stance, both feet aligned to a simple ground line, joints readable
