## Goal

Generate a public-safe 9 image pixel-art vehicles and interactables set for fast game ideation.

## Shared Prompt Rules

- Use case: stylized-concept
- Asset type: 2D pixel-art vehicle or interactive object resource
- Style: premium asset-store quality pixel art, crisp readable silhouette, hard edges, hand-placed pixel clusters, limited but polished palette
- Composition: one centered object per image, full object visible, generous padding
- Background: perfectly flat solid #ff00ff chroma-key background for local transparency removal
- Constraints: no text, no letters, no numbers, no watermark, no logo, no characters, no cast shadow, no contact shadow, no gradient background, do not use #ff00ff in the subject
- Output intent: keep a curated transparent PNG under `public/assets/vehicles/20260527-pixel-art-vehicles-interactables-pack-01/` and a feed-optimized WebP under `public/assets/feed-optimized/20260527-pixel-art-vehicles-interactables-pack-01/`
