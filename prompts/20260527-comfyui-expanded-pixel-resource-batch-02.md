# 20260527 ComfyUI Expanded Pixel Resource Batch 02

## Goal

Generate a broader set of 2D pixel game resource ideas that could plausibly appear in RPG, platformer, farming, crafting, adventure, and UI-heavy game prototypes.

## Scope

- Workflows: `qwen_image`, `pokemon`, `hidream_o1`
- Categories: 20 resource types from `tools/comfyui-resource-catalog-02.json`
- Target pack size: 9 images per workflow/category pack
- Total target: 540 generated PNG assets and 540 WebP feed previews

## Public-Safe Direction

All prompts should describe original, non-branded game resource concepts. Avoid readable text, logos, trademarks, and private references. Keep originals under `public/assets/` only after successful generation and publish feed media as optimized WebP files.
