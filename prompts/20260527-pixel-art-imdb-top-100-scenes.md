# 20260527-pixel-art-imdb-top-100-scenes

## Goal

Create 100 public-safe image-generation prompts that reinterpret famous scenes from IMDb Top 100-style classic movie entries as 16:9 cinematic pixel art illustrations.

## Source Basis

- Ranking basis: IMDb Top 250 Movies, using the first 100 titles from a public 2026 snapshot for prompt coverage.
- Exact rankings can change over time. Verify the live IMDb chart before using rank-dependent copy in public material.
- These prompts are reinterpretations, not requests to copy exact frames, actor likenesses, logos, title cards, posters, or trademark marks.

## Shared Prompt Rules

- Use case: stylized-concept
- Asset type: 2D pixel-art cinematic scene illustration
- Style: premium 16-bit / 32-bit pixel art, cinematic composition, crisp silhouettes, limited polished palette, readable values, subtle dithering, hand-placed pixel clusters, no painterly blur
- Composition: 16:9 wide scene, one iconic narrative moment, clear foreground/midground/background, strong lighting, no UI
- Public-safety constraints: no exact actor likeness, no exact frame recreation, no readable text, no letters, no numbers, no logos, no watermarks, no gore, no hate symbols, no explicit nudity
- Output intent: use as prompt notes only; no images are generated in this task

## Shared Negative Prompt

photorealistic, 3D render, smooth vector art, blurry, low resolution, unreadable muddy pixels, exact movie still, exact actor face, celebrity likeness, logo, watermark, poster text, subtitles, UI, frame border, gore, hate symbol
