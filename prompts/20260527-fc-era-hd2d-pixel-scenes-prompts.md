# 20260527-fc-era-hd2d-pixel-scenes-prompts

## Goal

Create a public-safe prompt set for reimagining the feeling of FC-era classic game moments as original HD-2D-inspired pixel-art scene illustrations.

## IP Safety Rule

Do not recreate any real game screenshot, named franchise, named character, mascot, boss, exact level map, cartridge art, title screen, logo, HUD, text, or trademarked symbol. Treat each entry as an original scene inspired by broad FC-era genre memories, not a copy of a specific game.

Use new character designs, altered silhouettes, original outfits, original enemies, changed layouts, and fresh color palettes.

## Shared Prompt Rules

- Use case: stylized-concept
- Asset type: 2D pixel-art key art / scene illustration
- Style: FC-era inspired pixel art reinterpreted with HD-2D-like modern rendering, 2.5D diorama depth, crisp pixel clusters, hand-placed pixel detail, rich but controlled palette
- Rendering: modern real-time game engine lighting, global illumination feel, volumetric fog, bloom, rim lighting, reflective wet surfaces where useful, layered parallax, depth of field on distant layers only
- Composition: cinematic scene, readable foreground/midground/background, strong focal point, no UI
- Output target: 16:9 scene illustration, high resolution, pixel-art look preserved
- Constraints: no text, no letters, no numbers, no watermark, no logo, no brand marks, no recognizable copyrighted characters

```text

```
