# 20260525-sideview-8x4-motion-sheets

## Goal

Generate four side-view semi-realistic 2D game motion sheets, one sheet per character.

Each sheet should use an `8 columns x 4 rows` layout:

1. Row 1: idle, 8 frames
2. Row 2: run, 8 frames
3. Row 3: attack, 8 frames
4. Row 4: jump, 8 frames

## Shared Prompt Constraints

- Use case: stylized-concept
- Asset type: side-view 2D game animation sprite sheet
- Style: non-pixel semi-realistic 2D digital painting, readable side-scroller game art
- Layout: one character only, 8 equal columns by 4 equal rows, 32 total frames
- Direction: every frame strict side profile facing right
- Motion: smooth frame-to-frame progression, consistent character proportions and costume across all frames
- Registration: feet and body anchored consistently for game slicing, character centered in each frame
- Avoid: front view, three-quarter view, mixed camera angles, pixel art, chibi proportions, 3D render, text labels, numbers, UI, logos, watermark, cropped limbs

## Motion Rows

- Row 1: idle breathing stance, subtle body bob and hand movement
- Row 2: run cycle, clear contact, passing, up, and down poses
- Row 3: primary weapon/tool attack, anticipation, swing/action, follow-through, recover
- Row 4: jump arc poses, crouch anticipation, takeoff, rise, apex, fall, landing
