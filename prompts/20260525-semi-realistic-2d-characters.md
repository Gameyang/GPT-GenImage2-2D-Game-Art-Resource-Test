# 20260525-semi-realistic-2d-characters

## Goal

Generate four non-pixel, semi-realistic 2D game character concepts as publishable character assets.

## Shared Prompt Constraints

- Use case: stylized-concept
- Asset type: 2D game character concept art
- Style: semi-realistic 2D digital painting, modern game art, clean readable silhouette
- Composition: full-body standalone character, centered, generous padding, neutral simple background
- Avoid: pixel art, chibi proportions, 3D render, anime-only style, heavy painterly blur, cropped limbs, text, logos, watermark

## Character Prompts

### 01 - Vanguard Swordswoman

Full-body standalone semi-realistic 2D game character concept of a disciplined vanguard swordswoman, practical leather-and-steel armor, long coat panels, one-handed sword at rest, confident ready stance, readable heroic silhouette, subtle battle wear, neutral studio background, no text, no logo, no watermark, not pixel art.

### 02 - Desert Ranger Archer

Full-body standalone semi-realistic 2D game character concept of a desert ranger archer, layered travel cloak, light armor, curved bow, utility belts and cloth wraps, alert survivalist pose, warm earth accents balanced with teal details, neutral studio background, no text, no logo, no watermark, not pixel art.

### 03 - Arcane Herbalist Mage

Full-body standalone semi-realistic 2D game character concept of an arcane herbalist mage, embroidered robe mixed with field gear, satchel of herbs and glass vials, wooden staff with subtle magical glow, calm focused pose, readable fantasy RPG silhouette, neutral studio background, no text, no logo, no watermark, not pixel art.

### 04 - Clockwork Shield Engineer

Full-body standalone semi-realistic 2D game character concept of a clockwork shield engineer, reinforced work jacket, brass-and-steel gauntlet, compact mechanical shield device, tool harness, sturdy defensive stance, believable functional equipment, neutral studio background, no text, no logo, no watermark, not pixel art.
